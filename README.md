# Multifilter search
## This is a small application thet implements multifilter search.
## Feature : 
- Search users by : 
    - name
    - username
    - email
## To use the app in local environment:

1. Go to  multifilter directory: 

    ```
    cd multifilter
    ```

2. intall dependencies: 
    ```
    npm install
    ```
3. run application:
    ```
    npm start
    ```
