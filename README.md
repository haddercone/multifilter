# Multifilter search
## This is a small application thet implements multifilter search.
## Feature : 
- Search users by : 
    - name
    - username
    - email
## To use the app in local environment:
1. Install dependencies
    ```
    npm install
    ```
2. Go to  multifilter directory: 

    ```
    cd multifilter
    ```

3. intall dependencies: 
    ```
    npm install
    ```
4. run application:
    ```
    npm start
    ```
